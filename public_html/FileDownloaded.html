Follow the instructions below to deploy the web application:

  1. 1) Accept the queued download file.

  2. 2) Double click on the file called "ffx.jnlp" located in the downloads folder

  3. 3) The file will be launched and you will be asked to install some of java components. Please accept and install all of these components.

  4. 4) The program will now begin and two separate windows will open. One window features the output of the function that you have chosen while the other window will provide a visual representation of the information stored in the pdb file that was uploaded.
